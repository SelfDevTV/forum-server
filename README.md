# Forum Server

A backend for a new forum application.

**How to use:**

---

- Clone the repo
- cd into the project
- `npm install`
- Create a `.env` file and provide it with a `PORT`, `DB_CONNECT` (Something like `DB_CONNECT=mongodb://localhost/forum`) and `TOKEN_SECRET`
- cd into `client`
- `npm install`
- cd back to the root (`cd ..`)
- `npm run dev`
